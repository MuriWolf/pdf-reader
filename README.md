# create-svelte
1. Clona o projeto pra sua máquina
2. vá para a pasta do projeto

# Comandos de inicialização do projeto
```bash
# Baixar dependencias
npm install 

# para entrar no VScode
code . 
```

# No VScode
Crie um arquivo .env.development
dentro dele coloque o valor que vou mandar


# rodar o site

```bash
npm run dev 
```
